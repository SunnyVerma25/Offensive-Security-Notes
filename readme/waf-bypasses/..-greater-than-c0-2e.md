# .. -> %c0%2e

../ is a common way to traverse directories. But if the user gets blocked by using ../ while attempting path traversal attacks, we can use the **Overlong UTF-8 dot format** to replace the .. and bypass WAF.

For example: The following image shows `../` gets blocked by Akamai WAF

<figure><img src="../../.gitbook/assets/image.png" alt=""><figcaption></figcaption></figure>

Replace the `..` with `%c0%2e%c0%2e`, and we can bypass the WAF and hit the Origin server!

<figure><img src="../../.gitbook/assets/image (221).png" alt=""><figcaption></figcaption></figure>

Replace the `..` with `%c0%ae%c0%ae` also works

<figure><img src="../../.gitbook/assets/image (220).png" alt=""><figcaption></figcaption></figure>

Use the following script to generate overlong utf-8 encoded characters of ascii characters: [https://github.com/ddittert/compute-overlong-utf8](https://github.com/ddittert/compute-overlong-utf8)
